== Description ==
This is a simple wordpress shoutbox plugin to display shoutbox in your widget, this shoutbox saves the history of shouts in database.

== Installation ==

This plugin follows the [standard WordPress installation method][]:

1. Upload the 'wordpress-simple-shout-box.zip' file to the `/wp-content/plugins/` directory using wget, curl of ftp.
2. 'unzip' the 'wordpress-simple-shout-box.zip' which will create the folder to the directory `/wp-content/plugins/wordpress-simple-shout-box` 
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Appearence -> Widgets -> Drag and Drop the "Qshout_widget" to your sidebar.
5. Save and Now you are done.
 
[standard WordPress installation method]: http://codex.wordpress.org/Managing_Plugins#Installing_Plugins
